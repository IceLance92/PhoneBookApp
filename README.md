# Телефонная адресная книга (Java, Android)

## Вход (создаются при первом запуске)
- admin / admin123
- user / user123

## Роли
- ADMIN: может создавать пользователей (USER) + их контакт, видеть все контакты, редактировать/удалять только свой контакт
- USER: может смотреть/искать контакты, редактировать/удалять только свой контакт

## Улучшения v1.2
- Фото контакта (выбор из галереи, сохраняется как URI)
- Импорт контактов из телефона (READ_CONTACTS, выбор нескольких)
- Сортировка: A→Z, Z→A, недавно добавленные
- Алфавитный индекс справа (быстрый переход к букве)
- Удаление контакта (только владельцем)
- Админ-список пользователей (только просмотр, поиск по логину)

## Важная проверка безопасности
Редактирование/удаление контакта возможно только владельцем:
`UPDATE/DELETE ... WHERE id = :contactId AND ownerUserId = :currentUserId`
